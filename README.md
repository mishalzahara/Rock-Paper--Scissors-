# Rock-Paper-Scissors Game

An interactive Rock-Paper-Scissors game built using HTML, CSS, and JavaScript. Players can compete against the computer with real-time result updates, score tracking, and a smooth, responsive user interface.

## Features

* Play Rock, Paper, or Scissors against the computer
* Random computer move generation
* Real-time game result display
* Live score tracking for player and computer
* Responsive and user-friendly design
* Smooth UI interactions and visual feedback
* Built with vanilla JavaScript (no external libraries)

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How to Play

1. Open the game in your browser.
2. Select Rock, Paper, or Scissors.
3. The computer will randomly choose its move.
4. The winner is determined based on the standard rules:

   * Rock beats Scissors
   * Scissors beats Paper
   * Paper beats Rock
5. Scores are updated automatically after each round.
